Asteroids

Prevent the asteroids from hitting your ship as you navigate through the Galaxy

Installation & Usage

Github Repo

# You will need to have Python installed on your system

$ git clone https://github.com/johnson-lee-v0/Asteroids.git
$ cd Asteroids
$ python3 Asteroid.py

Inspiration

I have always enjoyed playing retro style games and decided I wanted to create my own version of the famous asteroids.

Technologies

This game is made in python and the starting menu uses easygui

Future Ideas And Whats Next

Features that were considered but not built into the game includes adding adjustable diffuculty through
1) Adding time limit to each level
2) Limiting amount of ammunition that can be fired and add in a reload time
