Tetris

Replica of the 1985 Game

Installation & Usage

Github Repo

# You will need to have Python installed on your system

$ git clone https://github.com/johnson-lee-v0/Tetris.git
$ cd Tetris
$ python3 Tetris-1.py

Inspiration

I have always enjoyed playing retro style games and decided I wanted to create my own to test the limits of stacking.

Technologies

This game is made in python and the starting menu uses easygui

Future Ideas And Whats Next

Features that were considered but not built into the game
1) Storing Pieces so player would have more flexibilty
2) Tetris animation when four lines are cleared at the same time
