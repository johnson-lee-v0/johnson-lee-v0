Space Invaders

Can you fend off the Invaders before they take over your planet?

Installation & Usage

Github Repo

# You will need to have Java installed on your system

$ git clone https://github.com/johnson-lee-v0/SpaceInvaders.git
$ cd SpaceInvaders
$ java TitleScreen.java

Inspiration

One of the most iconic arcade games of all time, I wanted to create my own version to play!

Technologies

This game is made in Java and uses JFrame
